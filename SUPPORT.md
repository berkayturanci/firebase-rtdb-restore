# Support Policy

Thank you for using the **Firebase RTDB Lossless Restore Toolkit**! We want to ensure you have a great experience. Here is how you can get help or ask questions about this project.

---

## 💬 GitHub Discussions (Recommended)

For general questions, Q&A, setup assistance, or feature brainstorming, please use the **Discussions** tab:
* 👉 [GitHub Discussions](https://github.com/berkayturanci/firebase-rtdb-restore/discussions)

This is the best place to ask questions because other members of the community can benefit from the answers, and it keeps our Issue tracker focused strictly on actionable bugs and feature requests.

---

## 🐛 Bug Reports & Feature Requests

If you have confirmed a bug or want to request a new feature:
1. Search the [GitHub Issues](https://github.com/berkayturanci/firebase-rtdb-restore/issues) to see if it has already been reported.
2. If not, open a new issue using our interactive **Bug Report** or **Feature Request** forms.

---

## 🔒 Security Vulnerabilities

If you discover a security vulnerability, please do **NOT** open a public issue. Follow our [Security Policy](SECURITY.md) and report it privately via email to [berkayturanci@gmail.com](mailto:berkayturanci@gmail.com).
